# Design System MASTER

## 1. Overall Design Pattern
- **Pattern:** Dashboard
- **Layout Style:** Bento-Grid with Glassmorphism
- **Theme:** Dual-mode (Light/Dark)

## 2. Color Palette
- **Primary:** #0F62FE (Blue-700) / #4589FF (Blue-400)
- **Secondary:** #6F42C1 (Purple-700) / #A78BFA (Purple-300)
- **Surface (Light):** rgba(255,255,255,0.55)
- **Surface (Dark):** rgba(30,30,42,0.65)

## 3. Typography
- **Display:** Inter 700 / 2.5rem
- **Body:** Inter 400 / 1rem

## 4. Components
- **Card:** `.card { @apply bg-surface backdrop-blur-md border border-divider rounded-xl; }`
- **Status Badge:** Icon + Label + Background Tint
